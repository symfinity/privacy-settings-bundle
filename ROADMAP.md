## Details

### 0.1.x — Initial consent management

Strict privacy categories, consent capture, form-tier consent UI.

### 0.2.x — Policy integration

Privacy policies, consent versioning, policy references, consent history, migration support.

### 0.3.x — Cookie ecosystem

Cookie registry, service catalog, automatic categorization, provider metadata, consent dependencies.

### 0.4.x — Consent UX

Consent banner improvements, preference center, accessibility refinements, multilingual content, customization options.

### 0.5.x — Compliance tooling

Consent auditing, export formats, reporting, diagnostics, validation helpers, documentation.

### 0.6.x — Developer experience

CLI improvements, configuration helpers, Flex refinements, testing utilities, debugging tools, migration guides.

### 0.7.x — Integrations

Symfony Security integration, Twig improvements, JavaScript bridges, analytics adapters, third-party extension points.

### 0.8.x — Enterprise features

Multi-tenant support, organization defaults, shared policy management, deployment workflows, governance options.

### 0.9.x — Stabilization

API freeze, deprecation cleanup, performance improvements, documentation completion, cookbook, release preparation.

### 1.0.x — Stable API

Semver-stable extension points, stable configuration model, long-term compatibility guarantees, complete handbook.
